关于联想拯救者在ubuntu20.04未找到wifi适配器的问题

2021-11-15 ⏱ 约 3 分钟 Ubuntu20.04

序言

关于这个问题是我一直想总结的东西，但是苦于没有时间，总结的事情一直被搁置了。在11.15号英语口语课摸鱼中，心血来潮，便有了这篇vlog。

错误的尝试

sudo apt-get update
sudo apt-get install bcmwl-kernel-source

这是我打了很多遍的代码，不幸的是并没有什么用，在当时我还没搞清楚wifi适配器为啥找不到的时候，只能重启，输入，也没有找到别的解决方法hhh

sudo modprobe -r ideapad_laptop

移除ideapad无线模块，对我的电脑来说也没有用（难道是我的电脑没有用软碟通装Ubuntu的问题？）

对我电脑来说的正解

在明白是我没有安装无线网卡的问题之后，我也算从csdn上淘到了金子。（安装rtw89）解决方法如下：

1. 进入bios，将security boot设置为disenable，因为涉及到修改内核配置必须关掉安全启动。
2. 查询网卡状态（如果不一样需要重装）

lshw -C network

3. 下载驱动源码并安装

uname -sr
sudo apt-get update
sudo apt-get install make gcc linux-headers-$(uname -r) build-essential git
git clone https://github.com/lwfinger/rtw89.git

第一行是查询Ubuntu内核（小于5.4请先升级内核），第二第三行是必要的工具，最后安装驱动。
接着打开 rtw89/phy.c，注释掉：

bss_color = vif->bss_conf.he_bss_color.color;

再打开 rtw89/cam.c，注释掉：

u8 bss_color = vif->bss_conf.he_bss_color.color;
FWCMD_SET_ADDR_BSSID_BSS_COLOR(cmd, bss_color);
cd rtw89
make
cd /lib/firmware
mkdir rtw89
cd rtw89
sudo make install

若显示 install rtw89 SUCCESS 表明成功安装驱动。

4. 激活驱动

cd ~/rtw89
sudo modprobe rtw89pci

好了，恭喜我的电脑又出现了wifi图标！感谢观看。
